To use clock
1. Flip sw17 on to reset all values
2. Flip sw1 to enable set clock
3. Flip sw17 off
4. Use button key3 to increment hour
5. Use button key2 to increment minute
6. Flip sw1 down to retain set clock
7. Flip sw0 up to enable set alarm
8. Use button key1 to increment hour
9. Use button key0 to increment minute
10. Flip sw0 down to save alarm
11. Flip sw16 up to enable alarm
12. Flip sw16 down to disable alarm
